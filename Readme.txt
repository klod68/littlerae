Pequeño RAE
Este es un proyecto que realicé hace más de una década atrás y que he ido emigrando en cada versión nueva del .NET Framework. Ante mi uso diario de los diccionarios de la RAE me decidí a crear una simple aplicación Windows para usar los servicios Web del sitio, que se mantiene activa en el system tray o área de notificaciones.

Actualizaciones
Por ahora lo importante es mantener las direcciones a los servicios al día. Sería útil sacar las direcciones en un file de configuración para mantenerlo actualizado sin tener que cambiar la aplicación, pues ahora está integrada en el código...Sí ya sé, tonterías básicas, pero créanme, hice esto en poco tiempo y para ayudarme en la consulta rápida, no lo he tocado sino para cambiar la misma dirección de los servicios Web.
